Welcome to Workinglive's documentation for Authorization & Authentication
=========================================================================

The Workinglive Auth API provides a mechanism to authorize and authenticate. The API provide 
oath2.0 and openID Connect capabilities and describe how to configure your own services to access
the Workinglive service provider. 

Check out the :doc:`usage` section for further information, including how to
:ref:`configure <configuration>` your project.
